Codex 体感监测

感觉模型变笨了？别只靠感觉。

有时候 Codex 今天好像不太对劲：昨天能做的题，今天突然绕远；同样的问题，reasoning tokens 变少，答案也更容易错。LD-gpt-check 把这种体感变成可以记录的数据。

01

从玄学变成数据

记录正确率、reasoning tokens、输出 tokens、耗时和 TPS。

02

本地运行，可复现

直接调用你机器上的 Codex CLI，测试过程清晰透明。

03

社区共同观察

登录 Linux.do 后可以上传测试摘要，未来形成趋势图和榜单。

Features

一个给 AI 编程玩家准备的小工具

bin

单文件 CLI

下载即用，不需要折腾环境。

exec

自动调用 Codex

封装 codex exec --json，自动解析输出。

tok

推理 token 统计

不只看对错，也看模型到底有没有认真想。

ld

Linux.do 登录

支持网页登录和设备码登录，远程服务器也能用。

sum

可选上传结果

上传 summary，方便查看历史和社区趋势。

min

隐私克制

默认不上传本地数据库，不上传完整聊天历史。

How it works

它是怎么检测的？

1

本地调用 Codex CLI

在你的机器上执行 Codex，不代理、不托管你的请求。

2

输入固定糖果数学题

用稳定题目减少变量，让多次记录更容易对比。

3

解析 JSON 输出

读取 codex exec --json 的事件流，提取答案和 usage。

4

统计 token、耗时和正确率

把正确率、reasoning tokens、输出 tokens、TPS 放到同一张表里。

5

可选上传测试摘要

登录后上传 summary，为个人历史和社区趋势做准备。

Install

下载 CLI，然后跑一次。

Go 项目可以从源码安装，也可以等 GitHub Releases 提供预编译二进制。先确认本机已经安装并登录 Codex CLI。

$ go install github.com/haowang02/ld-gpt-check/cmd/ld-gpt-check@latest
$ ld-gpt-check run -m gpt-5.5 -r xhigh -n 5 --upload
