How it works

一条命令，三件事。

不做复杂后台，不做玄学评分。它只把一次本地检测跑完，然后把关键数字摆出来。

01

调用本机 Codex

封装 codex exec --json，测试过程仍然发生在你的机器上。

02

跑固定糖果题

同一道题重复测试，记录答案是否命中，方便横向和纵向对比。

03

输出 summary

正确率、reasoning tokens、输入输出 tokens、耗时和 TPS，一眼看完。

Features

核心功能就这些。

它是一个给 AI 编程玩家准备的小工具，重点是直接、可复现、少打扰。

local

本地运行

直接调用你机器上的 Codex CLI，不托管你的请求。

score

自动评分

固定题目、固定答案，把对错和正确率写清楚。

tokens

看 reasoning

不只看答没答对，也看模型到底花了多少 token 想。

share

可选上传

登录 Linux.do 后只上传 summary，后续可以做社区趋势。

Privacy

默认克制。

这个工具的目标是观察模型能力波动，不是收集你的本地数据。默认上传的是测试摘要，不是完整上下文。

不会上传本地 Codex 数据库
不会上传完整聊天历史
不会在客户端保存 OAuth secret

Install

下载，然后跑一次。

先确认本机已经安装并登录 Codex CLI，然后安装 LD-gpt-check。

$ go install github.com/haowang02/ld-gpt-check/cmd/ld-gpt-check@latest
$ ld-gpt-check run -m gpt-5.5 -r xhigh -n 5

Run it locally

现在跑一次体检。

一次测试不能定义模型，但一串记录可以看见趋势。
